# alpha-shop-react
- [AWS Amplify 展示連結](https://main.dbfkjg1s1o2x7.amplifyapp.com/)

## 刻意練習:
```
1. 從0建構整個購物車
2. 練習使用 useReducer + useContext 管理邏輯
3. 再由 useReducer + useContext 模式重構，改使用zustand 集中邏輯管理
4. 佈署至AWS Amplifyapp
```
---

## 本專案使用: 
```
react v18
fontawesome v5
zustand
eslint
prettier
```
---
## 專案功能
---
- 分成三個步驟完成購物車表單
- 調整所選商品數量
- 移除購物車商品
- 選擇運送的方式，會自動將運費導入
